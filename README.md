## 配置说明

### 记忆同步机制

每个 Agent 启动时执行：
1. 读取 `~/.openclaw/workspace/MEMORY.md` (共享规则)
2. 提取 `gz`/`jk` 标记的规则
3. 应用到当前会话
4. 读取本地配置文件

### 分层响应架构

**触发条件**: 任务预估耗时 > 2秒
**执行流程**:
```
用户请求 → 立即确认 → 创建子Agent → 执行 → 汇总回复
```

**适用场景**:
- 文件处理/数据分析
- 网络搜索/多步骤任务
- 复杂编码/长时间运行
- 深度研究报告
